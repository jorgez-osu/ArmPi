ArmPi Motion Flow – Pick / Sort / Stack

1. Initialize Arm
Move to home pose, reset gripper and wrist.
▼
2. Move Above Target Block
Use world (x, y) from perception, offset Y and height to hover above block.
▼
3. Orient Gripper
Rotate wrist based on block rotation angle so jaws align with block.
▼
4. Grasp Block
Open gripper, lower to grasp height, close gripper, then lift.
▼
5a. Place in Color Bin (Sorting)
Move above color-specific drop coordinate and place at fixed height.
or
5b. Place on Stack (Stacking)
Move above stack base, compute next stack height, place on top.
▼
6. Return to Home Pose
Lift, move back to initial pose, ready for next block.
